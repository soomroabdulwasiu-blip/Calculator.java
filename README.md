# Java Console Calculator

A simple console-based calculator built in Java to perform basic arithmetic operations. This project demonstrates core Java concepts such as variables, conditional statements, loops, switch-case, and user input handling

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Menu-driven program
- Uses switch-case and loops

## Concepts Used
- Variables and Data Types
- Scanner Class
- Conditional Statements
- switch-case
- do-while loop
- Basic error handling

## How to Run
1. Open terminal or command prompt
2. Compile the program:
   javac Calculator.java
3. Run the program:
   java Calculator

## Purpose
This project is created as a first Java project for GitHub to practice core Java programming concepts.
